# homebrew-gstack

Homebrew tap for the gstack tool family — AI-powered CLI tools extracted from [garrytan/gstack](https://github.com/garrytan/gstack).

## Install

```bash
brew tap juanheyns/gstack
```

### browse — persistent headless browser for AI agents

```bash
brew install browse
browse setup-chromium
```

### design — AI-powered UI mockup CLI

```bash
brew install design
design setup
```

## Upgrade

```bash
brew upgrade browse
brew upgrade design
```

## Tools

| Formula | Description | Source |
|---|---|---|
| [browse](https://github.com/juanheyns/gstack-browse) | Persistent headless browser CLI for AI agents | `juanheyns/gstack-browse` |
| [browse-agent](https://github.com/juanheyns/gstack-browse) | Claude AI sidebar add-on for browse | `juanheyns/gstack-browse` |
| [design](https://github.com/juanheyns/gstack-design) | AI-powered UI mockup generation via GPT-4o | `juanheyns/gstack-design` |
